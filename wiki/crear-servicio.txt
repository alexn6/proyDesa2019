Dentro del directorio del proyecto
En el archivo "/config/routes.yaml" agregar el servicio de la siguiente manera:

// esto para agregar el servicio y se pueda acceder desde afuera
=> testService:
    // como se llama el servicio
    path: /testSrvProy
    // controlador y funcion que devuelve los datos
    controller: App\Controller\testControllerSPT::respTestSrvTorneos



// conectar el controlador con el servicio
